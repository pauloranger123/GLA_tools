# GLA Tools

Ferramenta desktop (Windows) para ajudar líderes de aliança a gerenciar metas de doação (Berries e Gemas) dos membros, com interface temática inspirada no próprio jogo.

## Funcionalidades atuais

- Configuração inicial da quantidade de membros da aliança
- Criação de metas (cards) com prazo, valor de Berries e/ou Gemas
- Acompanhamento visual de quem já bateu a meta
- Bloqueio/desbloqueio de edição por meta (proteção contra edição acidental)
- Gerenciamento de membros (adicionar, renomear, remover)
- Dados salvos automaticamente em disco (não precisa de internet)

## Em desenvolvimento

- Calculadora de materiais de craft
- Histórico de metas passadas por membro

## Tecnologia

Desenvolvido em C# com WPF (.NET).
